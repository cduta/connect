# Connect

# Goal of the Game
Connect all so there are no open ends

# Controls
Movement       : Numpad (12346789) or Arrow Keys
Select/Deselect: Numpad 5 or Enter

n: Restart
u: Undo (up to 100 steps)
r: Redo
s: Save Game
l: Load Game
q: Quit
